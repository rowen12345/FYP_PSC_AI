Final Year Project
Conducted research under Erik Birgersson at NUS, developing AI models in MATLAB to predict current-voltage (IV) curves for perovskite solar cells using 31 input parameters.
• Reduced the number of predicted I–V curve points from 45 to 10 without compromising accuracy, significantly improving computational efficiency.
• Deployed Piecewise Cubic Hermite Interpolation (PCHIP) and spline interpolation to reconstruct the full I–V curve, achieving a mean RMSE of 5 based on over 6,600 predicted I–V curves. Applied convergence techniques to further enhance accuracy.
• Optimized neural network models with hyperparameter tuning (neurons, layers, activation functions, regularization), requiring 200 evaluations and 4-5 hours per model.
• Developed a MATLAB App Designer tool for users to input perovskite solar cell properties and predict IV curves in real time.
